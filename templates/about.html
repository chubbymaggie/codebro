What is CodeBro!?

CodeBro! is a Web-based code explorer using LLVM clang to parse C/C++ languages, and cross-referencing the whole source code (upstreaming and downstreaming) to enhance code lecture and easily (graphically?) detect bugs. CodeBro! is 100% Open-Source, so feel free to share, modify and fork it.

CodeBro! is a basically wrapper of the following kick-ass technologies:

  • clang + Python bindings, for code parsing;
  • Django, for the Web framework;
  • dajaxice, AJAX support for Django;
  • Pygments for code coloration;
  • Github Bootstrap to make Django look less ugly
